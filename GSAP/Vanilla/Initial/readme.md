4. O Pulo do Gato: toggleActions
   Essa é a parte que confunde muitos iniciantes. O valor "play none none none" significa:

onEnter: Ao entrar, dê play.

onLeave: Ao sair por cima, não faça nada (none).

onEnterBack: Ao voltar por cima, não faça nada.

onLeaveBack: Ao sair por baixo (scroll para cima), não faça nada.
