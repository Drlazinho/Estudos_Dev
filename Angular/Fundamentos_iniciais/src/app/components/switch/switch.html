Conteúdo por Papel (Uso do Switch no Angular)

@switch (userRole) { @case ('admin') {

Bem-vindo, administrador! Você tem acesso total ao sistema.

} @case ('editor') {

Bem-vindo, editor! Você pode editar conteúdos, mas não tem acesso às configurações do sistema.

} @default {

Bem-vindo, visitante! Você tem acesso limitado ao conteúdo público.

} }
